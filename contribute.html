How to contribute.
This magazine is in early state. 
If you want to contribute to it, please follow this rules:

1. Please contribute all your stuff in a single html file.
   If you want to contribute text only, write it into a 
   txt file and it will be edited. 

2. Images are not allowed right now.

3. each EXTERNAL link should have the target="_parent" tag inside.
    e.g. <a href="mysite.html" target="_parent">text</a>

4. Write to benobi@gmx.ch
